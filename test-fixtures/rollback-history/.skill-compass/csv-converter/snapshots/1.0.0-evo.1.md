# CSV Converter

## How to use

Read the CSV file using the **Read** tool. Parse headers from the first row.
Convert each row to a JSON object using headers as keys. Write output using the **Write** tool.

## Edge Cases

- Empty file: report error
- Missing headers: use column indices as keys
