# CSV Converter

## Prerequisites

- Input must be a CSV file (or TSV/custom delimiter)

## How to use

### Step 1: Read and detect format

Use the **Read** tool. Auto-detect delimiter (comma, tab, semicolon) and header presence.

### Step 2: Choose output

Ask: "Convert to JSON, YAML, or markdown table?"

### Step 3: Convert

**JSON**: Array of objects (if headers) or arrays
**YAML**: Same structure, YAML syntax
**Markdown**: Pipe-delimited table

### Step 4: Write

Use the **Write** tool.

## Edge Cases

- Empty file: report error
- Inconsistent columns: warn, pad with null
- Quoted fields with embedded delimiters: handle per RFC 4180
- Large files (>50MB): warn about memory
