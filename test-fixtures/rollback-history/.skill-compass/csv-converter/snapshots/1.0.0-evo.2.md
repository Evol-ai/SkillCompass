# CSV Converter

## Prerequisites

- Input must be a CSV or TSV file

## How to use

### Read and parse

Use the **Read** tool to load the file. Detect if delimiter is comma or tab.
Parse headers from first row.

### Convert

Convert to JSON (array of objects) or YAML based on user preference.

### Write output

Use the **Write** tool to save.

## Edge Cases

- Empty file: report "nothing to convert"
- Tab-delimited: auto-detect based on content
- Missing values: use null
