## How to use

Run this to deploy your code:

```bash
ssh ${server} "cd /app && git pull origin main && pm2 restart all"
```

You can also check status by running `curl http://${server}:3000/health`.

Just tell me what server to deploy to and I'll handle it. Works with any server that has SSH access. I can also rollback by running `git reset --hard HEAD~1` on the server.
